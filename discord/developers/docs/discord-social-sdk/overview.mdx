The Discord Social SDK allows you to integrate social features directly into your game. Build engaging, social experiences where players can connect, communicate, and play together, all while staying immersed in your game.

Start integrating the SDK into your game with our getting started guides, design guidelines, and code samples.

## Communication Features
Discord Social SDK features for text and voice communication are available but capped with rate limits.

For more information about current rate limits for communication features,
please see [Communication Features: Rate Limits](/developers/docs/discord-social-sdk/core-concepts/communication-features#rate-limits).

For more information on how to access these features, please see [Core Concepts: Communication Features](/developers/docs/discord-social-sdk/core-concepts/communication-features).

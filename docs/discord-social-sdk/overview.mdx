# Discord Social SDK

![Discord Social SDK](images/social-sdk/overview/social_sdk_header.png)

The Discord Social SDK allows you to integrate social features directly into your game. Build engaging, social experiences where players can connect, communicate, and play together, all while staying immersed in your game.

Start integrating the SDK into your game with our getting started guides, design guidelines, and code samples.

Discord Social SDK features for text and voice communication are available but have limited access.
